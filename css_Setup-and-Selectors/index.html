    CSS Setup and Selectors

    In this lesson you will learn how to set up a CSS file, and how to create CSS selectors, and how to set up CSS rule sets. 1. Intro to CSS * Cascading Style Sheets: is a language that web developers use to style the HTML content on a web page.

    1. Inline Styles 

      <p style="font-family: Arial;">I'm learning to code</p>

    2. The <style> Tag 

      <style>
p {
    color: red;
    font-size: 20px;
}
</style>

    3. The .css file

      • To avoid mixing code by storing HTML and CSS code in separate files.

    4. Linking the CSS file HTML <link href="./style.css" type="text/css" rel="stylesheet">

      • Use <link> element to link HTML and CSS diles togethor. Here are some arrtibutes:
      • href: like the anchor element, the value of this arrtibute must be the addresss, or path, to the CSS file.
      • type: The type of document that yout are linking to. The value of this attribute should be set to text/css.
      • rel: describes the relationship between the HTML dile and the CSS file. the value should be set to stylesheet.

    5. Tag Name 

      p {

}

      • CSS can select HTML elements by using and element's tag name.

    6. Class Name

      • HTML 
        <p class="brand">
      • CSS 
        .brand {

}
        It possible to select an element by its class( . ) attribute.

    7. Multiple Classes

      • HTML 
        <h1 class="title uppercase">
      • CSS 
        .title {
  color: teal;
}
.uppercase {
  text-transform: uppercase;
}
        It is able to add more than one class name to an HTML element's class attribute.

    8. ID Name

      • HTML 
        <h1 id="large-title">...</h1>
      • CSS 
        #large-title {

}
        An element needs to be styled uniquely.

    9. Classes and IDs

      1. Classes are meant to be used many times, an ID is meant to style only one element.
      2. IDs override the styles of tags and classes, So they should be used sparingly and only on elements that need to always appear the same.
      3. HTML 
        <h6 class="publish-time">Published: 2 Days Ago</h6>
      4. CSS 
        color: gray;

    10. Specificity

      1. Specificity is the order by which the browser decides which CSS styles will be displayed.
      2. In practice, using the lowest degree of specificity, so that if an element needs a new style, it is easy to override.
      3. tag > class > ID
      4. HTML 
        <h1 class="headline">Breaking News</h1>
      5. CSS 
        h1 {
    color: red;
}
.headline {
    color: firevrickl;
}

    11. Chaining Selectors

      1. it's possible to require an HTML element to have two or more CSS selectors at the same time. we will refer to as chaining. 
        h1.special {

}

    12. Nested Elements

      1. Nested elements can be selected by separating selectors with a space.
      2. HTML 
        <ul class='main-list'>
  <li> ...</li>
  <li> ...</li>
</ul>
      3. CSS 
        .main-list li{

}

    13. Chaing and Specificity

      1. Instead of selecting all <p> elements, you selected only the <p> elements nested inside the .main elements.
      2. CSS 
        p {
    color: blue;
}
.main p {
    color: red;
}
